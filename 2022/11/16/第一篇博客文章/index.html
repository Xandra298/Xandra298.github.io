hexo deploy 到github失败

使用deploy之后要求输入账户和密码，密码正确但是显示验证失败。

  • 失败原因：

    github 21年取消了密码验证

  • 解决方式

    在_config.yml的deploy中，repo对应的填写为github仓库的ssh地址，可以避免进行密码验证（前提：git配置完成，github上有设置公钥）
